In the notebooks included in this folder, you will find models for predicting the risk of heart disease and LDL cholesterol levels.

For a better understanding, start by analyzing the Python notebook. There you will find a descriptive analysis of the dataset used. 
We then analyze several prediction models to determine which one is best suited to our two problems.
You can then consult the R notebook, in which we also examine the different prediction models and compare them to the results obtained in Python.
At the end of this notebook, you will find the best methods for each prediction, as well as the recommended programming language.
